✨ 核心功能 (Features)

功能描述🚀 纯本地运行

单文件 HTML，0 依赖，无需联网，双击即用。

🎨 极简设计

现代化 UI，平滑动画，支持 深色模式 切换。

⚡ 智能组合

点击卡片自动组合 Tag，支持 (tag:1.1) 权重调整，一键复制。

🔒 数据隐私

所有数据仅存储在 浏览器本地 (Local Storage)，绝不上传服务器。

📂 一键备份

支持 JSON 格式的导入与导出，轻松迁移数据。

📖 快速上手 (Quick Start)

方法一：在线使用 (推荐)

直接访问 GitHub Pages 页面：

👉 点击这里开始使用

方法二：本地运行

如果你想断网使用，或者想修改代码：

点击本页面右上角的 Code -> Download ZIP。

解压文件。

直接双击 index.html 即可启动。

⚙️ 使用指南

1. 添加画师

点击右上角的 + 添加 按钮。

填写 显示名称 (例如 Mika Pikazo)。

填写 Tag (例如 mika pikazo)。

(可选) 上传一张本地图片作为封面。

点击保存。

2. 导出/备份数据

⚠️ 重要提示：因为数据保存在浏览器缓存中，清理缓存会导致数据丢失。

备份：点击顶部的 ⬇️ (下载图标)，保存 .json 文件。

恢复：点击顶部的 ⬆️ (上传图标)，选择备份文件即可恢复。
