🎨 画师 Prompt 管理器 (NovelAI Artist Manager)一个专为 AI 绘画爱好者设计的 本地化管理工具，帮助你高效整理、预览和组合画师标签 (Tags)。✨ 核心亮点🚀 纯本地运行：单文件 HTML，无需安装，无需联网，双击即用。🎨 极简设计：平滑动画、毛玻璃效果，适配深色模式。⚡ 智能组合：点击卡片即可组合关键词，支持权重调整，一键复制。🔒 数据安全：所有数据仅存储在您的浏览器本地，不会上传服务器。📖 快速上手教程1. 启动与界面打开方式：直接双击 index.html (原名 artist manager.html) 即可启动。界面布局：顶部：搜索、主题切换、导入/导出、添加按钮。左侧：画师卡片展示区。右侧：Prompt 生成区。2. 添加画师点击右上角的 + 添加 按钮。填写 显示名称 (如 "Mika Pikazo") 和 Tag (如 mika pikazo)。(可选) 选择一张本地图片作为封面。点击 保存。3. 组合与复制选择：点击左侧画师卡片，添加到生成列表。调整权重：点击 ^ 或 v 调整权重 (如 {tag:1.1}).复制：点击右下角的 复制 Tag 串 按钮。4. 数据备份 (重要!)⚠️ 注意：数据保存在浏览器缓存中，清理缓存会丢失数据。请养成定期备份的习惯。导出 (备份)：点击顶部的 ⬇️ (向下箭头)，保存 .json 文件。导入 (恢复)：点击顶部的 ⬆️ (向上箭头)，读取备份文件。🌐 在线版特别说明 (GitHub Pages)如果你访问的是部署在 GitHub Pages 上的版本：数据是私有的：你在网页上添加的数据，只保存在 你当前的浏览器 里。别人访问同一个网址，看到的是空白状态。如何分享：导出 JSON 文件发给朋友，让他们导入即可。❓ 常见问题 (Q&A)Q: 下次打开数据还在吗？A: 在的。 只要不清理浏览器缓存，数据会自动保存。Q: 图片显示不出来？A: 如果是本地部署，使用本地路径；如果是 GitHub Pages 在线版，建议上传图片到仓库或使用图床链接。Q: 既然是网页，我的数据会被上传吗？A: 不会。 本工具纯前端运行，没有任何后台服务器，数据绝对安全。<p align="center">Made with ❤️ for AI Art Community</p>
